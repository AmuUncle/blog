当下，Markdown 是一种广泛使用的轻量级标记语言，用于在Web中书写文档，同时应用于许多不同的场景。它旨在简化发布文本内容的过程，并减少在撰写文档时的格式转换的需要。Markdown 优点很多，包括易学易用、易于掌握和交流、跨平台可用性等等。

# Markdown 核心语法

1. 标题：通过使用 # 符号即可创建标题，例如：

```
# 一级标题
## 二级标题
### 三级标题
...
```

2. 斜体和加粗：可以使用 * 或 _ 符号将文本包裹起来实现，例如：

```
*斜体* 或者 _斜体_
**加粗** 或者 __加粗__
```

3. 引用：可以使用 > 符号引用文本，例如：

```
> 这是引用的文本。
```

4. 列表：可以通过符号 *, -, + 来创建无序列表，也可以使用数字来创建有序列表。例如：

```
- 无序列表项1
- 无序列表项2
1. 有序列表项1
2. 有序列表项2
```

5. 链接和图片：可以使用 [] 和 () 来创建链接和图片。例如：

```
[链接文字](链接地址)
![图片描述](图片地址)
```

6. 代码块：可以用 ``` 包裹一段代码创建代码块，例如：

```
​```javascript
console.log('Hello, world!');
​```
```

# Markdown 扩展语法

除了核心语法，Markdown 还有一些扩展语法来丰富文本格式。

1. 表格：可以使用竖线 | 和连字符 - 来创建表格，例如：

```
| 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 20   |
| 李四 | 22   |
```

2. 分割线：使用3个或3个以上的-、*、_可以创建分割线。

```
---
```

3. 脚注：可以使用[^]来创建脚注，例如：

```
这是一个脚注[^1]
[^1]: 这是脚注的文本。
```

4. 删除线：可以使用 ~~ 标记划掉的文本，例如：

```
~~删除的文本~~
```

以上就是常用的 Markdown 语法和扩展语法，掌握这些基础知识后就可以开始书写 Markdown 文档啦！
