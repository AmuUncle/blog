2021е№ҙ5жңҲ6ж—Ҙ дёӢеҚҲ
8.4k еӯ— 70 еҲҶй’ҹ ж¬Ў

гҖҗиҪ¬иҪҪгҖ‘LinuxepollжЁЎеһӢпјҲеҗ«LTжЁЎејҸе’ҢETжЁЎејҸиҜҰи§Јпјү

жң¬ж–ҮжңҖеҗҺжӣҙж–°дәҺпјҡ23 еҲҶй’ҹеүҚ

Linux epoll жЁЎеһӢпјҲеҗ«LT жЁЎејҸе’Ң ET жЁЎејҸиҜҰи§Јпјү

з»јеҗҲ select е’Ң poll зҡ„дёҖдәӣдјҳзјәзӮ№пјҢLinux д»ҺеҶ…ж ё 2.6 зүҲжң¬ејҖе§Ӣеј•е…ҘдәҶжӣҙй«ҳж•Ҳзҡ„ epoll жЁЎеһӢпјҢжң¬иҠӮжҲ‘д»¬жқҘиҜҰз»Ҷд»Ӣз»Қ epoll жЁЎеһӢгҖӮ

иҰҒжғідҪҝз”Ё epoll жЁЎеһӢпјҢеҝ…йЎ»е…ҲйңҖиҰҒеҲӣе»әдёҖдёӘ epollfdпјҢиҝҷйңҖиҰҒдҪҝз”Ё epoll_create еҮҪж•°еҺ»еҲӣе»әпјҡ

еҸӮж•° size д»Һ Linux 2.6.8 д»ҘеҗҺе°ұдёҚеҶҚдҪҝз”ЁпјҢдҪҶжҳҜеҝ…йЎ»и®ҫзҪ®дёҖдёӘеӨ§дәҺ 0 зҡ„еҖјгҖӮepoll_create еҮҪж•°и°ғз”ЁжҲҗеҠҹиҝ”еӣһдёҖдёӘйқһиҙҹеҖјзҡ„ epollfdпјҢи°ғз”ЁеӨұиҙҘиҝ”еӣһ -1гҖӮ

жңүдәҶ epollfd д№ӢеҗҺпјҢжҲ‘д»¬йңҖиҰҒе°ҶжҲ‘д»¬йңҖиҰҒжЈҖжөӢдәӢд»¶зҡ„е…¶д»– fd з»‘е®ҡеҲ°иҝҷдёӘ epollfd дёҠпјҢжҲ–иҖ…дҝ®ж”№дёҖдёӘе·Із»Ҹз»‘е®ҡдёҠеҺ»зҡ„ fd зҡ„дәӢд»¶зұ»еһӢпјҢжҲ–иҖ…еңЁдёҚйңҖиҰҒж—¶е°Ҷ fd д»Һ epollfd дёҠи§Јз»‘пјҢиҝҷйғҪеҸҜд»ҘдҪҝз”Ё epoll_ctl еҮҪж•°пјҡ

еҸӮж•°иҜҙжҳҺпјҡ

  • еҸӮж•° epfd еҚідёҠж–ҮжҸҗеҲ°зҡ„ epollfdпјӣ

  • еҸӮж•° opпјҢж“ҚдҪңзұ»еһӢпјҢеҸ–еҖјжңү EPOLL_CTL_ADDгҖҒEPOLL_CTL_MOD е’Ң EPOLL_CTL_DELпјҢеҲҶеҲ«иЎЁзӨәеҗ‘ epollfd дёҠж·»еҠ гҖҒдҝ®ж”№е’Ңз§»йҷӨдёҖдёӘе…¶д»– fdпјҢеҪ“еҸ–еҖјжҳҜ EPOLL_CTL_DELпјҢз¬¬еӣӣдёӘеҸӮж•° event еҝҪз•ҘдёҚи®ЎпјҢеҸҜд»Ҙи®ҫзҪ®дёә NULLпјӣ

  • еҸӮж•° fdпјҢеҚійңҖиҰҒиў«ж“ҚдҪңзҡ„ fdпјӣ

  • еҸӮж•° eventпјҢиҝҷжҳҜдёҖдёӘ epoll_event з»“жһ„дҪ“зҡ„ең°еқҖпјҢepoll_event з»“жһ„дҪ“е®ҡд№үеҰӮдёӢпјҡ

    epoll_event з»“жһ„дҪ“зҡ„ data еӯ—ж®өзҡ„зұ»еһӢжҳҜ epoll_data_tпјҢжҲ‘д»¬еҸҜд»ҘеҲ©з”ЁиҝҷдёӘеӯ—ж®өи®ҫзҪ®дёҖдёӘиҮӘе·ұзҡ„иҮӘе®ҡд№үж•°жҚ®пјҢе®ғжң¬иҙЁдёҠжҳҜдёҖдёӘ Union еҜ№иұЎпјҢеңЁ 64 дҪҚж“ҚдҪңзі»з»ҹдёӯе…¶еӨ§е°ҸжҳҜ 8 еӯ—иҠӮпјҢе…¶е®ҡд№үеҰӮдёӢпјҡ

  • еҮҪж•°иҝ”еӣһеҖјпјҡepoll_ctl и°ғз”ЁжҲҗеҠҹиҝ”еӣһ 0пјҢи°ғз”ЁеӨұиҙҘиҝ”еӣһ -1пјҢдҪ еҸҜд»ҘйҖҡиҝҮ errno й”ҷиҜҜз ҒиҺ·еҸ–е…·дҪ“зҡ„й”ҷиҜҜеҺҹеӣ гҖӮ

еҲӣе»әдәҶ epollfdпјҢи®ҫзҪ®еҘҪжҹҗдёӘ fd дёҠйңҖиҰҒжЈҖжөӢдәӢд»¶е№¶е°ҶиҜҘ fd з»‘е®ҡеҲ° epollfd дёҠеҺ»еҗҺпјҢжҲ‘д»¬е°ұеҸҜд»Ҙи°ғз”Ё epoll_wait жЈҖжөӢдәӢд»¶дәҶпјҢepoll_wait еҮҪж•°зӯҫеҗҚеҰӮдёӢпјҡ

еҸӮж•°зҡ„еҪўејҸе’Ң poll еҮҪж•°еҫҲзұ»дјјпјҢеҸӮж•° events жҳҜдёҖдёӘ epoll_event з»“жһ„ж•°з»„зҡ„йҰ–ең°еқҖпјҢиҝҷжҳҜдёҖдёӘиҫ“еҮәеҸӮж•°пјҢеҮҪж•°и°ғз”ЁжҲҗеҠҹеҗҺпјҢevents дёӯеӯҳж”ҫзҡ„жҳҜдёҺе°ұз»ӘдәӢд»¶зӣёе…і epoll_event з»“жһ„дҪ“ж•°з»„пјӣеҸӮж•° maxevents жҳҜж•°з»„е…ғзҙ зҡ„дёӘж•°пјӣtimeout жҳҜи¶…ж—¶ж—¶й—ҙпјҢеҚ•дҪҚжҳҜжҜ«з§’пјҢеҰӮжһңи®ҫзҪ®дёә 0пјҢepoll_wait дјҡз«ӢеҚіиҝ”еӣһгҖӮ

еҪ“ epoll_wait и°ғз”ЁжҲҗеҠҹдјҡиҝ”еӣһжңүдәӢд»¶зҡ„ fd ж•°зӣ®пјӣеҰӮжһңиҝ”еӣһ 0 иЎЁзӨәи¶…ж—¶пјӣи°ғз”ЁеӨұиҙҘиҝ”еӣһ -1гҖӮ

epoll_wait дҪҝз”ЁзӨәдҫӢеҰӮдёӢпјҡ

epoll_wait дёҺ poll зҡ„еҢәеҲ«

йҖҡиҝҮеүҚйқўд»Ӣз»Қ poll дёҺ epoll_wait еҮҪж•°зҡ„д»Ӣз»ҚпјҢжҲ‘д»¬еҸҜд»ҘеҸ‘зҺ°пјҡ

epoll_wait еҮҪж•°и°ғз”Ёе®Ңд№ӢеҗҺпјҢжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘеңЁ event еҸӮж•°дёӯжӢҝеҲ°жүҖжңүжңүдәӢд»¶е°ұз»Әзҡ„ fdпјҢзӣҙжҺҘеӨ„зҗҶеҚіеҸҜпјҲevent еҸӮж•°д»…д»…жҳҜдёӘеҮәеҸӮпјүпјӣиҖҢ poll еҮҪж•°зҡ„дәӢд»¶йӣҶеҗҲи°ғз”ЁеүҚеҗҺж•°йҮҸйғҪжңӘж”№еҸҳпјҢеҸӘдёҚиҝҮи°ғз”ЁеүҚжҲ‘д»¬йҖҡиҝҮ pollfd з»“жһ„дҪ“зҡ„ events еӯ—ж®өи®ҫзҪ®еҫ…жЈҖжөӢдәӢд»¶пјҢи°ғз”ЁеҗҺжҲ‘д»¬йңҖиҰҒйҖҡиҝҮ pollfd з»“жһ„дҪ“зҡ„ revents еӯ—ж®өеҺ»жЈҖжөӢе°ұз»Әзҡ„дәӢд»¶пјҲ еҸӮж•° fds ж—ўжҳҜе…ҘеҸӮд№ҹжҳҜеҮәеҸӮпјүгҖӮ

дёҫдёӘз”ҹжҙ»дёӯзҡ„дҫӢеӯҗпјҢжҹҗдәәдёҚж–ӯз»ҷдҪ дёҖдәӣиӢ№жһңпјҢиҝҷдәӣиӢ№жһңжңүз”ҹжңүзҶҹпјҢи°ғз”Ё epoll_wait зӣёеҪ“дәҺпјҡ

и°ғз”Ё poll зӣёеҪ“дәҺпјҡ

еҪ“з„¶пјҢиҝҷе№¶дёҚж„Ҹе‘ізқҖпјҢpoll еҮҪж•°зҡ„ж•ҲзҺҮдёҚеҰӮ epoll_waitпјҢдёҖиҲ¬еңЁ fd ж•°йҮҸжҜ”иҫғеӨҡпјҢдҪҶжҹҗж®өж—¶й—ҙеҶ…пјҢе°ұз»ӘдәӢд»¶ fd ж•°йҮҸиҫғе°‘зҡ„жғ…еҶөдёӢпјҢepoll_wait жүҚдјҡдҪ“зҺ°еҮәе®ғзҡ„дјҳеҠҝпјҢд№ҹе°ұжҳҜиҜҙ socket иҝһжҺҘж•°йҮҸиҫғеӨ§ж—¶иҖҢжҙ»и·ғиҝһжҺҘиҫғе°‘ж—¶ epoll жЁЎеһӢжӣҙй«ҳж•ҲгҖӮ

LT жЁЎејҸе’Ң ET жЁЎејҸ

дёҺ poll зҡ„дәӢд»¶е®ҸзӣёжҜ”пјҢepoll ж–°еўһдәҶдёҖдёӘдәӢд»¶е®Ҹ EPOLLETпјҢиҝҷе°ұжҳҜжүҖи°“зҡ„иҫ№зјҳи§ҰеҸ‘жЁЎејҸпјҲEdge TriggerпјҢETпјүпјҢиҖҢй»ҳи®Өзҡ„жЁЎејҸжҲ‘д»¬з§°дёә ж°ҙе№іи§ҰеҸ‘жЁЎејҸпјҲLevel TriggerпјҢLTпјүгҖӮиҝҷдёӨз§ҚжЁЎејҸзҡ„еҢәеҲ«еңЁдәҺпјҡ

  • еҜ№дәҺж°ҙе№іи§ҰеҸ‘жЁЎејҸпјҢдёҖдёӘдәӢд»¶еҸӘиҰҒжңүпјҢе°ұдјҡдёҖзӣҙи§ҰеҸ‘пјӣ
  • еҜ№дәҺиҫ№зјҳи§ҰеҸ‘жЁЎејҸпјҢеҸӘжңүдёҖдёӘдәӢд»¶д»Һж— еҲ°жңүжүҚдјҡи§ҰеҸ‘гҖӮ

иҝҷдёӨдёӘиҜҚжұҮжқҘиҮӘз”өеӯҰжңҜиҜӯпјҢдҪ еҸҜд»Ҙе°Ҷ fd дёҠжңүж•°жҚ®и®ӨдёәжҳҜй«ҳз”өе№іпјҢжІЎжңүж•°жҚ®и®ӨдёәжҳҜдҪҺз”өе№іпјҢе°Ҷ fd еҸҜеҶҷи®ӨдёәжҳҜй«ҳз”өе№іпјҢfd дёҚеҸҜеҶҷи®ӨдёәжҳҜдҪҺз”өе№ігҖӮйӮЈд№Ҳж°ҙе№іжЁЎејҸзҡ„и§ҰеҸ‘жқЎд»¶жҳҜзҠ¶жҖҒеӨ„дәҺй«ҳз”өе№іпјҢиҖҢиҫ№зјҳжЁЎејҸжҳҜзҠ¶жҖҒж”№дёәй«ҳз”өе№іпјҢеҚіпјҡ

ж°ҙе№іжЁЎејҸзҡ„и§ҰеҸ‘жқЎд»¶

иҫ№зјҳжЁЎејҸзҡ„и§ҰеҸ‘жқЎд»¶

иҜҙзҡ„жңүзӮ№жҠҪиұЎпјҢд»Ҙ socket зҡ„иҜ»дәӢд»¶дёәдҫӢпјҢеҜ№дәҺж°ҙе№іжЁЎејҸпјҢеҸӘиҰҒ socket дёҠжңүжңӘиҜ»е®Ңзҡ„ж•°жҚ®пјҢе°ұдјҡдёҖзӣҙдә§з”ҹ POLLIN дәӢд»¶пјӣиҖҢеҜ№дәҺиҫ№зјҳжЁЎејҸпјҢsocket дёҠз¬¬дёҖж¬Ўжңүж•°жҚ®дјҡи§ҰеҸ‘дёҖж¬ЎпјҢеҗҺз»ӯ socket дёҠеӯҳеңЁж•°жҚ®д№ҹдёҚдјҡеҶҚи§ҰеҸ‘пјҢйҷӨйқһжҠҠж•°жҚ®иҜ»е®ҢеҗҺпјҢеҶҚж¬Ўдә§з”ҹж•°жҚ®жүҚдјҡз»§з»ӯи§ҰеҸ‘гҖӮеҜ№дәҺ socket еҶҷдәӢд»¶пјҢеҰӮжһң socket зҡ„ TCP зӘ—еҸЈдёҖзӣҙдёҚйҘұе’ҢпјҢдјҡдёҖзӣҙи§ҰеҸ‘ POLLOUT дәӢд»¶пјӣиҖҢеҜ№дәҺиҫ№зјҳжЁЎејҸпјҢеҸӘдјҡи§ҰеҸ‘дёҖж¬ЎпјҢйҷӨйқһ TCP зӘ—еҸЈз”ұдёҚйҘұе’ҢеҸҳжҲҗйҘұе’ҢеҶҚдёҖж¬ЎеҸҳжҲҗдёҚйҘұе’ҢпјҢжүҚдјҡеҶҚж¬Ўи§ҰеҸ‘ POLLOUT дәӢд»¶гҖӮ

socket еҸҜиҜ»дәӢд»¶ж°ҙе№іжЁЎејҸи§ҰеҸ‘жқЎд»¶пјҡ

socket еҸҜиҜ»дәӢд»¶иҫ№зјҳжЁЎејҸи§ҰеҸ‘жқЎд»¶пјҡ

socket еҸҜеҶҷдәӢд»¶ж°ҙе№іжЁЎејҸи§ҰеҸ‘жқЎд»¶пјҡ

socket еҸҜеҶҷдәӢд»¶иҫ№зјҳжЁЎејҸи§ҰеҸ‘жқЎд»¶пјҡ

д№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңеҜ№дәҺдёҖдёӘйқһйҳ»еЎһ socketпјҢеҰӮжһңдҪҝз”Ё epoll иҫ№зјҳжЁЎејҸеҺ»жЈҖжөӢж•°жҚ®жҳҜеҗҰеҸҜиҜ»пјҢи§ҰеҸ‘еҸҜиҜ»дәӢд»¶д»ҘеҗҺпјҢдёҖе®ҡиҰҒдёҖж¬ЎжҖ§жҠҠ socket дёҠзҡ„ж•°жҚ®ж”¶еҸ–е№ІеҮҖжүҚиЎҢпјҢд№ҹе°ұжҳҜдёҖе®ҡиҰҒеҫӘзҺҜи°ғз”Ё recv еҮҪж•°зӣҙеҲ° recv еҮәй”ҷпјҢй”ҷиҜҜз ҒжҳҜEWOULDBLOCKпјҲEAGAIN дёҖж ·пјүпјӣеҰӮжһңдҪҝз”Ёж°ҙе№іжЁЎејҸпјҢеҲҷдёҚз”ЁпјҢдҪ еҸҜд»Ҙж №жҚ®дёҡеҠЎдёҖж¬ЎжҖ§ж”¶еҸ–еӣәе®ҡзҡ„еӯ—иҠӮж•°пјҢжҲ–иҖ…ж”¶е®ҢдёәжӯўгҖӮиҫ№зјҳжЁЎејҸдёӢж”¶еҸ–ж•°жҚ®зҡ„д»Јз ҒзӨәдҫӢеҰӮдёӢпјҡ
